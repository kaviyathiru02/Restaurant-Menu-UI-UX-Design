# Restaurant-Menu-UI-UX-Design
A modern Restaurant Menu mobile app UI/UX design created in Figma with interactive prototype navigation.

## Features 
Welcome Screen,
Food Categories Screen,
Food Items Screen,
Cart Screen,
Order Confirmation Screen,
Interactive Prototype Navigation.

## Screens
Welcome Screen,
Categories Screen,
Food Items Screen,
Cart Screen,
Order Confirmation Screen.

## Tools Used
Figma,
Unsplash Images,
Iconify Icons.

## Prototype Flow
Welcome → Categories → Food Items → Cart → Order Confirmation.
